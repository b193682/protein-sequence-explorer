# Protein Sequence explorer

A web-based bioinformatics tool using protein sequence information retrieved from the NCBI datatbase using Entrez EDirect command line tools. 
Users can;
- Retrieve sequences from specific protein families and taxonomic groups
- Perform MSA using Clustal Omega
- Analayse conservation across species
- Discover relevant protein structures from the PDB
- Analyse pairwise identity heatmaps and summary statistics
- Identify motifs using the PROSITE database (EMBOSS)
- View protein characteristics such as amino acid identities using EMBOSS tools
